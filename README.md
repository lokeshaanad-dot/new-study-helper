# AI Study Helper (OpenEnv Hackathon)

This project is an AI-powered study assistant built for the OpenEnv RL Hackathon.

## Features
- Provides simple and clear study explanations
- Uses OpenAI API for generating responses
- Lightweight and efficient for limited compute environments

## Project Structure
- inference.py: Main inference script (used for evaluation)
- app.py: Hugging Face Space interface
- requirements.txt: Dependencies

## Environment Variables
- API_BASE_URL (default provided)
- MODEL_NAME (default provided)
- HF_TOKEN (required)

## Notes
- Designed to run within 2 CPU and 8GB RAM constraints
- Output format follows OpenEnv submission guidelines
